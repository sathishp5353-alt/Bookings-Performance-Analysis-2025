# Bookings-Performance-Analysis-2025
## Project Description
This project analyzes 20,000 booking records using SQL and Power BI
to understand revenue trends, customer behavior, and cancellations.

## Tools Used
- Excel
- MYSQL
- Power BI

**END-TO-END PROJECT FLOW**

CSV File
 ↓
MySQL (SQL Analysis)
 ↓
Power BI (Dashboard + DAX)
 ↓
Business Insights

## Key KPIs
- Total Bookings
- Total Revenue
- Average Booking Value
- Cancellation Rate

## Outcome
Created an interactive Power BI dashboard for business insights.

**SQL PROJECT QUESTIONS**
1.Show all bookings made in 2025
2.Count total number of bookings
3.Find total revenue from all bookings
4.List all distinct cities
5.Show bookings where status = Cancelled
6.Find bookings with amount greater than 10,000
7.Total bookings by service_type
8.Total revenue by city
9.Average booking amount per service
10.Count bookings by payment_mode
11.Month-wise total bookings
12.Month-wise total revenue
13.Find top 5 cities by revenue
14.Find total cancelled bookings by city
15.Calculate cancellation rate (%)
16.Find customers with more than 5 bookings
17.Rank cities based on revenue
18.Find peak booking month
19.Identify service type with highest cancellation
20.Running total of revenue month-wise

**POWER BI**
I designed a multi-page Power BI dashboard with KPIs, trends,
and drill-downs using DAX measures, slicers, and interactive visuals to support business decisions.
